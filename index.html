UW-Madison - CS559 - Project 3 - Graphics Town

Author: Brian Ploeckelman and James Pulec

This program uses several libraries, and requires some environment variables to be set in order to build correctly.

Fltk - http://www.fltk.org/ SFML 1.6 - http://sfml-dev.org GLM 0.9.3 - http://glm.g-truc.net

Environment variables:

FLTKHOME - the root fltk directory SFML_1_6 - the root sfml 1.6 directory SFML_1_6_LIBS - the sfml 1.6 lib file directory GML - the root glm 0.9.3 directory
